## Configuration

<img align="center" src="https://github.com/christianalfoni/signalit/assets/3956929/eb74b1ea-0ff1-4d18-9ba5-97150408ae86" width="25"/> **Babel**

```bash

```

```json
{
    "plugins": [

    ]
}
```

This is a **Stage 3** proposal and is coming to JavaScript. It is also available in TypeScript from version 5.2.
